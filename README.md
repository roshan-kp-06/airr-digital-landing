# Airr Digital Landing Page

Performance marketing agency landing page with a bold, editorial brutalist aesthetic.

## Features

- Fully responsive design
- Smooth scroll animations
- Counter animations for statistics
- Modern dark theme with chartreuse accents
- Grain texture overlay for visual depth
- Intersection Observer for scroll-triggered animations

## Tech Stack

- Pure HTML, CSS, and JavaScript
- No frameworks required
- Google Fonts (Syne + DM Sans)

## Development

```bash
npx serve .
```

## Deployment

Optimized for Vercel deployment. Simply connect your GitHub repo to Vercel for automatic deployments.
